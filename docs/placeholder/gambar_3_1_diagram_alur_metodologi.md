# Gambar 3.1: Diagram Alur Metodologi Penelitian

## Deskripsi
Diagram flowchart yang menunjukkan alur metodologi penelitian dari awal hingga akhir.

## Elemen yang Harus Ditampilkan

```
1. Pengumpulan Data (OAI-PMH Harvesting)
        ↓
2. Eksplorasi Data Awal (EDA Raw Data)
        ↓
3. Pembersihan Data
        ↓
4. Eksplorasi Data Bersih (EDA Clean Data)
        ↓
5. Preprocessing Teks
        ↓
6. Pemodelan LDA
        ↓
7. Evaluasi Model
        ↓
8. Visualisasi & Dashboard
```

## Catatan
- Gunakan bentuk kotak/persegi panjang untuk setiap tahap
- Gunakan panah untuk menunjukkan alur
- Bisa menggunakan warna berbeda untuk setiap fase utama
