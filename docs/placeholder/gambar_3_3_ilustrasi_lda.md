# Gambar 3.3: Ilustrasi Konsep Model LDA

## Deskripsi
Diagram yang mengilustrasikan konsep Latent Dirichlet Allocation.

## Elemen yang Harus Ditampilkan

1. **Korpus Dokumen** (kumpulan dokumen)
2. **Topik** (representasi probabilistik kata-kata)
3. **Distribusi Topik per Dokumen** (setiap dokumen = campuran topik)
4. **Distribusi Kata per Topik** (setiap topik = campuran kata)

## Contoh Ilustrasi

```
Dokumen 1: [Topik A: 60%, Topik B: 30%, Topik C: 10%]
Dokumen 2: [Topik A: 10%, Topik B: 70%, Topik C: 20%]

Topik A: [kata1: 0.3, kata2: 0.2, kata3: 0.15, ...]
Topik B: [kata4: 0.25, kata5: 0.2, kata6: 0.1, ...]
```

## Catatan
- Bisa menggunakan plate notation (diagram grafis LDA standar)
- Atau ilustrasi sederhana dengan dokumen, topik, dan kata
